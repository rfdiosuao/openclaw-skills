# Thinking Skill - 深度思考与规划

> 让行动前先思考成为习惯

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://clawhub.ai/skills/thinking-skill)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

---

## 🎯 Skill 定位

**元认知思考引擎** - 让 Agent 在行动前先思考、检索记忆、整合经验、规划最佳路径

---

## ⚡ 快速开始

### 安装

```bash
clawhub install thinking-skill
```

### 触发词

- `/thinking` - 标准触发
- `先思考一下` - 中文触发
- `think about this` - 英文触发
- `规划一下` - 规划触发

### 使用示例

```
/thinking 帮我分析一下最近的股票走势，然后生成一份报告发到飞书群里
```

---

## 🏗️ 核心思考流程

### 6 步思考法

```
┌─────────────────────────────────────────┐
│  1. 任务解析  — 理解真实需求             │
│  2. 技能盘点  — 确认可用工具             │
│  3. 记忆检索  — 查找相关经验             │
│  4. 路径规划  — 设计最优方案             │
│  5. 风险评估  — 预判问题和应对           │
│  6. 开始执行  — 按规划行动               │
└─────────────────────────────────────────┘
```
