# openclaw-onboarding

OpenClaw 新手智能体快速配置工具 - 根据目标角色自动安装必备平台和技能

## 🎯 功能

- **平台安装** - 自动安装 ClawHub、SkillHub、虾评 Skill CLI
- **技能检测** - 强制先安装 Skill Vetter 检测工具
- **角色导向** - 根据用户目标（内容创作者/编程助手/数据分析等）推荐技能
- **智能搜索** - 自动搜索并安装相关技能
- **配置向导** - 交互式引导新手配置

## 🚀 使用方法

```
配置内容创作者智能体
设置编程助手
我要做数据分析智能体
新手 OpenClaw 配置
```

## 📦 安装顺序

1. **平台 CLI** → ClawHub + SkillHub + 虾评
2. **检测工具** → Skill Vetter（强制）
3. **核心技能** → 根据角色搜索安装
4. **扩展技能** → 相关辅助技能

## 🎭 支持的角色

| 角色 | 关键词 |
|------|------|
| 内容创作者 | 写作、发帖、视频 |
| 编程助手 | 编程、开发、代码 |
| 数据分析 | 数据、分析、报表 |
| 客服助手 | 客服、问答、支持 |
| 办公助手 | 日程、会议、文档 |
| 营销助手 | 营销、SEO、推广 |
